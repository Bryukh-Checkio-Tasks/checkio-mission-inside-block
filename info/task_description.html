We learned how to target and shoot walls in a previous mission . Now we must determine if a bullet will hit a specific shaped target. The target is a polygon, which is represented as a list of vertex coordinates on a plane. Vertices are connected sequentially with the last vertex in the list connecting to the first. We have the coordinates of the point on the plane that the bullet struck. If the bullet's point of impact lies on the edge of the polygon the hit is true. For this mission, you need to determine whether the point of impact lies inside the target.

Input: A list of lists for the vertices coordinates and impact point coordinates. Coordinates are presented as a list of two integers. [[x1, y1], [x2, y2], [x3, y3]], [xp, yp]]

Output:A boolean. True (hits the target) or False (misses the target.)

Example: 
checkio([[[1,1],[1,3],[3,3],[3,1]], [2,2]]) == True
checkio([[[1,1],[1,3],[3,3],[3,1]], [4,2]]) == False
checkio([[[1,1],[4,1],[2,3]], [3,2]]) == True
checkio([[[1,1],[4,1],[1,3]], [3,3]]) == False
checkio([[[2,1],[4,1],[5,3],[3,4],[1,3]], [4,3]]) == True
checkio([[[2,1],[4,1],[3,2],[3,4],[1,3]], [4,3]]) == False
checkio([[[1,1],[3,2],[5,1],[4,3],[5,5],[3,4],[1,5],[2,3]], [3,3]]) == True
checkio([[[1,1],[1,5],[5,5],[5,4],[2,4],[2,2],[5,2],[5,1]], [4,3]]) == False

How it is used: Images recognizing.
